# Appendix: Glossary

- `Source of truth`: The single authoritative editable file for a concern.
- `Generated artifact`: File produced by build tooling; not manually edited.
- `Anti-flail`: Required sequence preventing blind iterative breakage.
- `Hypothesis`: The single expected causal effect of one change.
- `Rollback trigger`: Observable condition requiring immediate reversion.
- `Validation matrix`: Structured checklist proving behavior and regression safety.
- `potch`: Project-level progress/change record.
- `Canonical`: The currently authoritative policy or file.
